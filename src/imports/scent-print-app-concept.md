Design a high-fidelity mobile app concept for a product called “Scent Print”.

CORE SYSTEM:
One unified design system with automatic Light and Dark Mode switching (like iOS).
Same layout and components across modes.
Only theme tokens adapt.

This is NOT a perfume app.
It is a sensory memory system for capturing and reconstructing scent experiences.

--------------------------------------------------
VISUAL DIRECTION — Warm Human Spatial Interface
--------------------------------------------------

Style:
Minimal, warm, human-centered, calm.
Editorial and refined.
Emotionally grounded.

Scent Visualization:
Layered atmospheric mist density field.
Soft warm gradient core.
Subtle volumetric light.
Very restrained micro-particles.
Feels like air illuminated by natural light.

Motion:
Slow breathing cycle (4–5 seconds).
Gentle expansion and contraction.
Soft glow pulse in Dark Mode.
Even softer diffusion in Light Mode.

--------------------------------------------------
THEME TOKENS
--------------------------------------------------

Dark Mode:
Deep warm-neutral background (#0E0F12 range)
Copper / muted amber accent
Subtle volumetric glow
No pure black

Light Mode:
Warm gray background (#F4F2EF range)
Soft neutral surfaces
Same copper accent adjusted for contrast
No pure white

--------------------------------------------------
NAVIGATION
--------------------------------------------------

Bottom tab bar:
Capture
Archive
Map
Share

Ultra minimal typography.
Thin sliders.
Soft floating capture button.

--------------------------------------------------
SCREENS
--------------------------------------------------

1) Capture
Live scent mist visualization
Primary action: “Capture”
Microcopy: “Hold near the source for 6–10 seconds.”

2) Encoding
Progress ring
Label: “Encoding Scent Print…”

3) Save Snapshot
Fields: Title, Tags, Notes
Auto metadata: Time, GPS Location
CTA: “Save as Snapshot”

4) Archive
Minimal grid of Scent Snapshots
Each has small mist thumbnail
Search + filters

5) Snapshot Detail
Large mist visualization
Controls: Intensity, Radius, Duration
Buttons: Replay, Share

6) GLOBAL SCENT MAP (Important)

Full-screen interactive world map.
Zoomable and draggable (like Apple Maps).
User location indicator included.
Subtle grayscale or desaturated map style.

Instead of normal pins:
Use soft halo diffusion rings to mark scent locations.
Each location shows a warm subtle glow.
Tapping a halo opens a floating preview card with:
- Snapshot name
- Mini mist visualization
- “Replay”
- “Open”

Include:
- Current location button (bottom right)
- Zoom controls (minimal)
- Smooth map scrolling and pinch-to-zoom

The map should feel calm and warm, not busy.
No bright map markers.

7) Share
Select contact
Add message
Privacy toggle: Private / Public link
CTA: “Send Scent Print”

8) Receive
Incoming scent preview
Buttons: Replay Now, Save to Archive, View on Map

--------------------------------------------------
IMPORTANT RULES
--------------------------------------------------

No e-commerce layouts.
No bright multicolor gradients.
No decorative fireworks particles.
Keep spacing generous.
Maintain warm human tone.
Make the interface feel emotionally grounded and spatial.
